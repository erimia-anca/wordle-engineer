## ✦ Wordle Engineer

This project was created as a team project. My main contributions were developing the HTML structure, styling the application with CSS, and implementing the game logic using JavaScript.

The project is a Wordle-inspired game where players have six attempts to guess a five-letter word. It also includes a leaderboard system powered by Firebase Firestore, allowing players to save their scores and compare their results with others.

### ✦ About the project

Wordle Ingineresc is a browser-based word guessing game inspired by the original Wordle game.

The player has six attempts to guess a randomly selected five-letter word. After each attempt, the letters are visually marked depending on whether they are in the correct position, exist in the word, or are not part of the word.

The project also includes a leaderboard system powered by Firebase Firestore. After successfully guessing the word, players can enter their name and save their score based on the number of attempts used.

## ✦ The process

The idea for the project came from observing how many technical terms and words in French can appear while studying at a technical university. To make learning these words more fun and interactive, we decided to create a Wordle-inspired game focused on French vocabulary.

The project started with creating the basic game structure and understanding how the Wordle guessing system works. The game board was then generated dynamically using JavaScript, with six rows and five boxes for each possible guess.

The application then connects to Firebase Firestore, where the available five-letter words are stored. When the player enters a guess, the game compares each letter with the selected word and changes the tile colors depending on the result.

After the main game logic was completed, animations were added to make the feedback more interactive. The leaderboard was then implemented using Firestore, allowing players to save their name and number of attempts after successfully guessing the word.
